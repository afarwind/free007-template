
# 自用webpack+vue项目模板

> 基于nodejs开发的前端项目，依赖nodejs开发环境

#### 编译过程

1. 使用 npm

``` bash

# install dependencies
npm install

# serve with hot reload at localhost:5700
npm run dev

# build for production with minification
npm run build


```

2.使用yarn（推荐，需自行安装yarn）

``` bash

# install dependencies
yarn install

# serve with hot reload at localhost:5700
yarn run dev

# build for production with minification
yarn run build


```
