> 基于 nodejs 开发的前端项目，依赖 nodejs 开发环境

#### 编译过程

1. 使用 npm

```bash

# install dependencies
npm install

# serve with hot reload at localhost:5700
npm run dev

# build for production with minification
npm run build


```

2.使用 yarn（推荐，需自行安装 yarn）

```bash

# install dependencies
yarn install

# serve with hot reload at localhost:5700
yarn run dev

# build for production with minification
yarn run build


```
